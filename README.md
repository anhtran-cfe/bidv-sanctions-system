- ⚙️ Batch Processing

- 📊 Real-time Dashboard

## Setup

1. Get Gemini API key from [Google AI Studio](https://aistudio.google.com/app/apikey)

2. Configure in Streamlit Cloud secrets: GEMINI_API_KEY = "your_key"

3. Deploy and enjoy!
